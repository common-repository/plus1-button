== Description ==
<h2>Stable: 1</h2>
<p>
Nice shortcode for Google +1 dead simple to use
[plus1 count="true" size="standard"]

<strong>Size Examples: </strong>
Tall
[plus1 count="true" size="tall" ]

medium:
[plus1 count="true" size="medium"]

Small:
[plus1 count="true" size="small"]

You can also set the url to the +1 by wrapping the short code around a url
[plus1 count="true" size="small"]http://exmaple.com[/plus1]
</p>

== Installation ==

1. Upload the `plus1-button` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. For a Example use got to <a href="http://gfxcomplex.com/wordpress_plugin/google-1-easy-button-wordpress-plugin/">http://gfxcomplex.com/wordpress_plugin/google-1-easy-button-wordpress-plugin/</a> or view the screenshot
